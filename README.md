## Development set up

Clone Repository using
`git clone git@github.com:mrEckendonk/jest-module-testing.git`

OR using HTTPS

`https://github.com/mrEckendonk/jest-module-testing.git`

move into project directory
`cd jest-module-testing`

## Run tests

`npm test`
